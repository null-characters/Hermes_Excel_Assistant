# Excel 处理技能规范

## 技能名称
excel_processor

## 描述
处理 Excel 文件的各种操作，包括数据清洗、汇总、分析、格式转换等。

## 适用场景
- 数据清洗：去除重复、处理空值、格式标准化
- 数据汇总：按条件统计、透视表、合并多表
- 数据分析：趋势分析、异常检测、基础统计
- 格式转换：列重命名、类型转换、表格格式化

## 技术规范

### 库使用优先级
1. **pandas** - 首选，用于数据处理
2. **openpyxl** - 用于样式操作和复杂 Excel 格式
3. **xlrd/xlwt** - 仅用于旧版 .xls 文件

### 代码规范

```python
from datetime import datetime

# ... 业务逻辑 ...
```

### 格式规范

| 类型 | 规范 | 示例 |
|------|------|------|
| 日期格式 | YYYY-MM-DD | 2026-05-14 |
| 时间格式 | HH:MM:SS | 14:30:00 |
| 数值精度 | 保留2位小数 | 1234.56 |
| 百分比 | 保留1位小数 | 85.5% |
| 金额 | 千分位分隔 | 1,234,567.89 |

### 输出文件命名
- 处理后文件后缀: `_processed`
- 示例: `销售数据.xlsx` → `销售数据_processed.xlsx`

## 错误处理

1. **文件不存在**: 返回明确错误信息
2. **格式不支持**: 提示支持的格式
3. **数据异常**: 记录异常行，继续处理
4. **内存不足**: 分块处理大文件


## 安全要求

- 禁止执行任何文件系统操作（删除、移动系统文件）
- 禁止网络请求
- 仅处理传入的 Excel 文件
- 输出文件仅保存到指定目录
