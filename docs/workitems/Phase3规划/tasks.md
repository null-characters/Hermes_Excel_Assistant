## 任务编排原则

1. **安全优先**: T03-01 ~ T03-04 必须最先完成（P0）
2. **功能迭代**: 批量处理 → 模板 → 预览，按依赖顺序执行
3. **质量并行**: T03-20 ~ T03-23 可与功能开发并行
4. **测试驱动**: 每个功能模块完成后立即编写测试
